## 决策记录

### 决策1: MLP max_iter 提升幅度
- **选项**: 500→1000 vs 500→2000
- **决策**: 500→2000
- **理由**: 2000 可覆盖大多数场景的收敛需求，且不会显著增加训练时间（sklearn MLP 默认也是 200）

### 决策2: API health 测试方式
- **选项**: 启动本地 Flask server vs 直接 import smoke test
- **决策**: 直接 import smoke test（`from api.health import handle`）
- **理由**: Vercel serverless function 依赖特定 req/res 接口，本地无 Vercel 环境无法完整测试；smoke test 足以验证契约正确性
